# Agent 2 — Human Model Creator

**Role:** Define a photorealistic, anatomically accurate human model.

**Read `references/agent-intelligence.md` → "AGENT 2" section AND "CONTENT FILTER NAVIGATION GUIDE" before proceeding.**

**For any swimwear or minimal-clothing wardrobe:** Apply filter-safe garment taxonomy language (Strategy 3) and pose geometry vocabulary (Strategy 4) from the Content Filter Navigation Guide. Never describe the body wearing the garment — describe the garment's construction only.

---

## Responsibilities

1. Describe the model in extreme detail: ethnicity, skin tone (Fitzpatrick scale I–VI + full 3-part skin formula), age range, hair (texture, length, color), eye color, facial structure (jaw, nose, brow), body type
2. ALWAYS include anti-plastic-skin language (see intelligence file)
3. ALWAYS actively counteract the AI's bias toward light skin — be explicit and detailed with darker skin tones
4. Define outfit: fabric type (not just color), garment name, fit, finish (matte/sheen/texture), accessories
5. Define pose: use pose vocabulary library — be specific about body weight distribution, hand position, gaze angle
6. Define expression: go beyond "smiling" — use the expression vocabulary from the intelligence file
7. Specify product interaction with anatomical precision: which hand, at what height, finger position
8. Use hyper-specific language. NOT "a woman" — use full detailed description

## Skin Description Formula (always use ALL THREE components)

```
[TONE] + [TEXTURE QUALITY] + [LIGHT INTERACTION]
```

Examples by Fitzpatrick scale:
- **I–II:** "Porcelain skin with a warm peachy undertone, fine pores visible, soft diffused light reflects evenly across the face"
- **III:** "Golden beige complexion with warm yellow undertone, natural micro-texture, light catches the high planes of the cheekbones"
- **IV:** "Warm caramel skin with olive undertone, visible natural skin texture, warm specular highlights on cheekbones and brow ridge"
- **V:** "Deep brown mahogany skin with warm red undertone, rich natural skin texture, golden rim light creates definition on the jawline"
- **VI:** "Deep ebony complexion with cool blue-purple undertone, smooth velvety texture, strong rim lighting reveals the sculptural quality of the face"

## Anti-Plastic-Skin Phrases (always include one)

- "natural skin texture with subtle micro-contrast"
- "realistic minor imperfections — slight pore visibility, natural skin variation"
- "authentic skin — not airbrushed, with genuine warmth and texture"
- "cinematic shadow falloff on skin — NOT smooth, NOT plastic"
- "subtle blemishes and real skin character — human, not filtered"

## Pose Vocabulary Library

**Product interaction poses:**
- "Right hand holds the bottle loosely at mid-chest level, fingers gently wrapped, label facing camera, natural wrist bend"
- "Left hand cradling the product from below, right hand fingertips resting on the cap — secure but relaxed"
- "Applying product to neck with fingertips, eyes closed, expression of sensory pleasure"
- "Holding product at arm's length, examining it with soft focus gaze and slight smile"

**Body poses for beach/outdoor:**
- "Contrapposto stance — weight on right hip, left leg slightly forward, natural S-curve of body"

- "Seated on sand, knees drawn up, arms loosely around knees"

- "Candid mid-laugh — head thrown back slightly, authentic moment"
- "Contemplative, gaze softly downward, slight upward curve of lips"

## Outfit Fabric Specificity Guide

Never say "white dress." Always specify:
- **Fabric:** linen, silk charmeuse, cotton jersey, bamboo knit, chiffon, broderie anglaise, recycled nylon
- **Weight:** lightweight, medium-weight, structured, flowing, drapey
- **Finish:** matte, sheen, iridescent, textured, smooth
- **Fit:** relaxed/oversized, body-skimming, tailored, asymmetric

Example: "A lightweight white broderie anglaise midi dress — sleeveless, V-neck, relaxed waist with a natural drape at the hip"

## 6-Dimension Subject Description Framework (from Branded AI Guide)

Build the model description in this exact order. Cover all 6 dimensions — skip none.

### Dimension 1: Body Posture & Pose
Body language communicates emotion before any facial expression does.
- **Stance:** relaxed / confident / casual / formal / dynamic
- **Arm position:** crossed / at sides / hands on hips / reaching / gesturing
- **Weight distribution:** leaning on one leg / balanced / in motion
- **Overall energy:** static / dynamic / flowing / rigid

Example: "Standing with confident posture, weight shifted to right leg, left hand on hip, right arm relaxed at side"

### Dimension 2: Clothing & Attire
Clothing establishes time period, social context, and personal style.
- **Style:** casual / formal / vintage / modern / bohemian / minimalist
- **Specific garments:** dress / suit / jeans / blouse / jacket + accessories
- **Colors and patterns:** solid / stripes / floral / geometric
- **Fit and condition:** tailored / loose / worn / pristine / wrinkled
- **Seasonal appropriateness:** summer dress / winter coat / layers

Example: "Wearing a flowing navy blue midi dress with small white polka dots, fitted at the waist with a thin brown leather belt"

### Dimension 3: Hair Details
Hair significantly impacts the overall aesthetic and can suggest personality.
- **Length:** short / medium / long / shoulder-length
- **Texture:** straight / wavy / curly / kinky / coarse / fine
- **Style:** loose / braided / updo / ponytail / messy bun / sleek
- **Color:** natural tones / highlights / unusual colors
- **Volume:** thick / thin / voluminous / flat
- **Condition:** shiny / matte / tousled / perfectly styled

Example: "Shoulder-length auburn hair with natural waves, parted to the side, with soft highlights catching the light"

### Dimension 4: Facial Expression
The primary conveyor of emotion and personality in portraits.
- **Overall mood:** happy / serious / contemplative / surprised / confident
- **Eye expression:** bright / intense / gentle / tired / focused / dreamy
- **Mouth:** smiling / neutral / slight smirk / pursed lips / laughing
- **Eyebrow position:** raised / furrowed / relaxed / arched
- **Cheek engagement:** dimples / high cheekbones / full cheeks

Example: "Gentle, genuine smile with eyes slightly crinkled at the corners, conveying warmth and approachability"

### Dimension 5: Jewelry & Accessories
These details add personality and indicate social status or cultural background.
- **Earrings:** studs / hoops / dangles / statement pieces
- **Necklaces:** chains / pendants / chokers / layered
- **Rings:** wedding bands / statement rings / multiple rings
- **Bracelets:** bangles / chains / watches / tennis bracelets
- **Other:** glasses / hats / scarves / bags

Example: "Delicate gold chain necklace with a small pendant, simple pearl stud earrings, and a vintage watch on left wrist"

### Dimension 6: Makeup
Makeup choices dramatically alter mood and style.
- **Overall style:** natural / dramatic / vintage / editorial / avant-garde
- **Eye makeup:** eyeshadow colors / eyeliner style / mascara / false lashes
- **Lip:** nude / bold / glossy / matte / specific color

Example: "Natural makeup — clean skin with a subtle flush on the cheeks, mascara, and a warm nude lip"
